# Phone Book

Phone Book is a (super) simple website that allows you to store contact information.

## Requirements

You're gonna need:
- Ruby
- Sinatra

## Setup

1. Clone this repository.
1. Launch the app.
1. Enjoy!

```
$ git clone https://github.com/zulaica/phone_book.git
$ ruby app.rb
```
